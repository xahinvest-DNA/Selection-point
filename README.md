# Selection Point

**Selection Point** — самостоятельная система восстановления способности выбора.

## Текущее состояние

- Активная фаза: **Фаза 3 — точная архитектура пяти ступеней**.
- Активная ступень: **Ступень 1 «Жизнь внутри автоматизма»**.
- Утверждены параметры SP-S1-P01–SP-S1-P12.
- **SP-S1-P13 «Ловушка ступени» не открыт.**

Актуальная контрольная точка:

- `docs/FOUNDATION/CURRENT_PROJECT_STATE.md`;
- `docs/CURRENT_STATE.md` — короткая совместимая точка входа.

## Последнее утверждённое положение

**SP-S1-P12 — «Критерии перехода».**

Рабочее название:

> **«От различимого эпизода к сопоставимым эпизодам».**

Утверждённая формула:

> **Переход начинается не тогда, когда автоматизм исчез, а когда отдельные повторения стали различимыми, изменяемыми и пригодными для сопоставления.**

Переход обоснован, когда человек повторяемо и с ограниченной структурной поддержкой:

- локализует несколько конкретных автоматических эпизодов;
- реконструирует их как функционально связанные последовательности;
- различает факт и гипотезу;
- реалистично оценивает доступность хода;
- влияет на часть продолжений или следующих циклов;
- сохраняет эпизоды как отдельные сопоставимые единицы.

Для перехода не требуются исчезновение автоматизма, предотвращение первой реакции, полная самостоятельность, перенос во все сферы или внешний предметный успех.

Полный источник:

- `docs/FOUNDATION/FIVE_STAGES_THEORY/12_STAGE_1_TRANSITION_CRITERIA.md`.

Канонический источник:

- `docs/FOUNDATION/CANONICAL/04_STAGE_1_P12_CANONICAL.md`.

## Граница с Фазой 6

В Фазе 3 SP-S1-P12 определяет качественный профиль готовности.

В Фазе 6 будут отдельно утверждаться:

- количество эпизодов;
- временной период;
- допустимый объём помощи;
- способ фиксации;
- процедура оценки;
- пороги;
- исключения.

Источник:

- `docs/FOUNDATION/TRANSITION_CRITERIA_BOUNDARY.md`.

## Сквозная визуальная модель

SP-VM-01 «Граф траекторий выбора» связана с SP-S1-P05–SP-S1-P12.

Для P12 несколько отдельных последовательностей:

```text
позиция → ход → следующая позиция
```

могут быть сохранены как сопоставимые эпизоды. Способ их сравнения относится к следующей ступени и пока не утверждён.

## Что сейчас нельзя делать

- не открывать SP-S1-P13 без отдельного указания Андрея;
- не превращать качественный профиль P12 в числовую систему раньше Фазы 6;
- не считать внешний результат доказательством способности;
- не считать реконструкцию ведущего способностью участника;
- не утверждать архитектуру второй ступени;
- не создавать конкретные практики раньше Фазы 5;
- не проектировать курс, пилот, приложение или технический стек.

> Не скорость генерации, а последовательность утверждённых решений создаёт архитектуру Selection Point.
