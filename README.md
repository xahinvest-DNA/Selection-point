# Selection Point

**Selection Point** — mobile-first система внутренней трансформации через тело, энергию и жизнь.

Цель v1: создать приложение для телефона, которое помогает человеку не “садиться на диету”, а видеть фактическую структуру своего поведения, формулировать внутреннюю задачу и проходить короткие циклы действия с обратной связью.

## Базовая продуктовая гипотеза

Первая версия фокусируется не на трейдинге, а на более быстро проверяемой области:

- здоровое и энергичное тело;
- снижение лишнего веса без войны с собой;
- питание, сон, движение, энергия, алкоголь, пищевые импульсы;
- связь телесных привычек с общей жизненной управляемостью.

## Роли

- **Андрей** — владелец продукта и главный источник направления.
- **ChatGPT** — главный менеджер проекта: структура, приоритеты, декомпозиция, контроль логики, задания для Codex.
- **Codex** — исполнитель: реализация кода, тесты, рефакторинг, технические решения в рамках зафиксированной архитектуры.

## Формат продукта

Базовый формат: **мобильное приложение**.

Рекомендуемый стек MVP: Expo + React Native + TypeScript.

Причина: быстрый старт, Android/iOS, возможность локального прототипа без раннего backend, удобная работа с Codex.

## Главная механика

```
Факт → Потенциал → Разность потенциалов → Внутренняя задача → Практика → Действие → Обратная связь → Новый цикл
```

## MVP

Первый MVP — 14-дневный протокол **“Тело без войны”**:

1. День 0: диагностика.
2. Дни 1–3: честная карта текущего состояния.
3. Дни 4–7: выбор одного главного механизма.
4. Дни 8–11: система вместо силы воли.
5. Дни 12–14: обратная связь и новая внутренняя задача.
