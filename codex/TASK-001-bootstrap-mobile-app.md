## Почему задача неактивна

Задача была создана в период, когда мобильное приложение, 14-дневный протокол и стек Expo + React Native рассматривались как уже выбранное направление.

Действующая архитектура устанавливает, что:

- формат первой версии определяется только в Фазе 7;
- до цифрового продукта проводится ручной пилот в Фазе 8;
- приложение, AI, данные и технический стек проектируются только в Фазе 9;
- технология должна обслуживать утверждённый процесс, а не задавать его заранее.

Следовательно, сейчас не утверждены:

- необходимость мобильного приложения;
- mobile-first подход;
- Expo;
- React Native;
- TypeScript как продуктовый стек;
- набор экранов;
- навигация;
- модель данных;
- local-first хранение;
- отсутствие или наличие backend;
- конкретный пользовательский loop.
