# CURRENT_STATE

**Статус:** совместимая точка входа.  
**Авторитетный динамический документ:** `docs/FOUNDATION/CURRENT_PROJECT_STATE.md`.

Этот файл не ведётся как самостоятельный источник истины. Он существует для совместимости со старыми ссылками и направляет читателя к актуальной архитектуре.

## Актуальная контрольная точка

- Активная фаза: **Фаза 3 — точная архитектура пяти ступеней**.
- Активная ступень: **Ступень 1 «Жизнь внутри автоматизма»**.
- Утверждены параметры SP-S1-P01–SP-S1-P06:
  1. состояние человека;
  2. способ восприятия реальности;
  3. способ восприятия себя;
  4. что управляет решениями;
  5. доступная точка выбора;
  6. главная иллюзия — «Иллюзия неизбежного продолжения».
- Следующий вопрос: **SP-S1-P07 — главная боль**.

## Последнее утверждённое положение

> **Главная иллюзия первой ступени состоит в том, что текущая внутренняя сборка ситуации переживается как объективная необходимость.**

Короткая формула:

## Утверждённая визуальная модель

SP-VM-01 «Граф траекторий выбора»:

```text
текущая позиция
→ поле доступных ходов
→ следующий ход
→ новая позиция
→ новое поле ходов
```

Главная формула:

> **Каждый ход создаёт поле следующего хода.**

Источники:

- `docs/FOUNDATION/VISUAL_MODELS/01_CHOICE_TRAJECTORY_GRAPH.md`;
- `docs/FOUNDATION/VISUAL_MODELS/assets/choice_trajectory_graph.svg`.

Модель не означает равной доступности, безопасности или обратимости всех направлений и не объявляет реальные ограничения только внутренней иллюзией.

## Архитектура знаний

Действуют четыре основных слоя:

```text
полная теория
→ каноническая архитектура
→ общие библиотеки
→ материалы доставки
```

Визуальные модели являются сквозным вспомогательным слоем представления и не заменяют ни один из четырёх основных слоёв.

После каждого утверждения обязательно создаются полный файл, каноническая карточка и обновления библиотек. Новый полный файл применяет правило дельты.

## Что сейчас не утверждено

До соответствующих фаз нельзя считать принятыми решениями:

- главную боль первой ступени;
- следующий навык;
- конкретные практики на основе графа траекторий;
- количество модулей и уроков;
- 14-дневный формат;
- формат курса;
- приложение и AI-сопровождение;
- технический стек, backend, хранилище, авторизацию и платежи.

## Следующее действие

Продолжать работу только с SP-S1-P07 — «главная боль» — без смешения со следующим навыком, практиками, критериями перехода или ловушкой ступени.
