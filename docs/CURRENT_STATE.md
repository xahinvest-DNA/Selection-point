# CURRENT_STATE

**Статус:** совместимая точка входа.  
**Авторитетный динамический документ:** `docs/FOUNDATION/CURRENT_PROJECT_STATE.md`.

Этот файл больше не ведётся как самостоятельный источник истины. Он существует для совместимости со старыми ссылками и должен направлять читателя к актуальной архитектуре.

## Актуальная контрольная точка

- Активная фаза: **Фаза 3 — точная архитектура пяти ступеней**.
- Активная ступень: **Ступень 1 «Жизнь внутри автоматизма»**.
- Утверждены параметры 1–4:
  1. состояние человека;
  2. способ восприятия реальности;
  3. способ восприятия себя;
  4. что управляет решениями.
- Следующий вопрос: **параметр 5 — доступная точка выбора**.

Полное и обновляемое состояние находится в:

- `docs/FOUNDATION/CURRENT_PROJECT_STATE.md`;
- `docs/FOUNDATION/PROJECT_OPERATING_PROTOCOL.md`;
- `docs/FOUNDATION/FIVE_STAGES_THEORY/00_INDEX.md`.

## Принцип приоритета

При любом расхождении между этим файлом, ранними продуктовыми документами и материалами в `docs/FOUNDATION/` приоритет имеют документы `docs/FOUNDATION/`.

## Что сейчас запрещено считать утверждённым

До соответствующих фаз нельзя считать принятыми решениями:

- 14-дневный протокол;
- фиксированное количество уроков;
- готовую карту модулей курса;
- mobile-first формат;
- приложение;
- AI-сопровождение;
- Expo, React Native или иной стек;
- backend, хранилище, авторизацию и платежи.

## Следующее действие

Продолжать работу только с параметра 5 первой ступени — «доступная точка выбора» — без смешения с главным навыком, главной иллюзией, главной болью, практиками или критериями перехода.
