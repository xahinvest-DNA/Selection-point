# Карта перекрёстных связей Selection Point

**Статус:** действующая навигационная библиотека  
**Дата обновления:** 21 июля 2026 года

Карта показывает, какие утверждения являются основанием для следующих элементов и где искать полный смысл.

## Переход от первой ступени ко второй

```text
несколько отдельно реконструированных эпизодов
→ эпизоды сохранены как сопоставимые единицы
→ человек замечает, что некоторые повторения не случайны
→ внешнее сходство ещё смешивается с функциональной организацией
→ SP-S2-P01
```

SP-S1-P12 передаёт второй ступени сопоставимый материал. SP-S1-P13 защищает от ярлыка, объяснения и превосходства вместо участия.

## Центральные причинные линии

### SP-HCM-09 и SP-VM-01

```text
текущая позиция
→ ход или бездействие
→ следующая позиция
→ реальность следующего цикла
```

### SP-S2-P02

```text
сценарное восприятие
→ знакомое значение и ожидание
→ знакомый ход
→ знакомая следующая позиция
→ подтверждение сценарной реальности
```

### SP-S2-P04

```text
сценарная роль
→ что необходимо сохранить, доказать или не допустить
→ сценарная задача
→ оперативная задача
→ приоритетный ход
```

### SP-S2-P05

```text
сценарное требование обнаружено
≠ сценарное требование обязательно к исполнению
```

### SP-S2-P06

```text
Пространство 1 переживается как нежелательное
→ Пространство 2 строится как противоположность
→ дистанция от нежелательного принимается за движение к желаемому
```

### SP-S2-P08

```text
фактическая следующая позиция
→ что повторилось?
→ что изменилось?
→ какая функция сохранилась?
→ как скорректировать следующий доступный ход?
```

### SP-S2-P10

```text
жизненная ситуация
→ знакомая роль
→ сценарная задача
→ распределение времени, денег,
  внимания, близости, риска и обязательств
→ действие или бездействие
→ материальная и социальная следующая позиция
→ новое поле ходов
```

### SP-S2-P11

```text
ограниченная область исследования
→ несколько функционально сопоставимых циклов
→ рабочая сценарная гипотеза
→ различение текущего сценарного требования
→ один реально доступный проверяемый ход
→ фактическая следующая позиция
→ уточнение или опровержение гипотезы
→ корректировка следующего доступного хода
```

### SP-S2-P12

```text
проверяемый цикл становится повторяемо доступным
→ сценарий перестаёт быть единственным организатором
→ открывается вопрос:
что я сам создаю дальше?
```

### SP-S2-P13

```text
сценарий различим
→ возникает требование больше никогда не быть прежним собой
→ создаётся противоположная роль
  или обязательное Пространство 2
→ следующий ход должен доказать освобождение
→ результат становится экзаменом идентичности
→ сценарий снова организует следующий цикл
```

## Связь SP-S2-P11 → SP-S2-P13

```text
SP-S2-P11:

SP-S2-P12:

```

## Центральные утверждённые различения

> **Контроль предполагает возможность заранее получить желаемый результат. Создание происходит независимо от контроля.**

> **Мне не обязательно этим ходом доказывать, кто я.**

> **Противоположность нежелательному — ещё не выбранное направление.**

> **Возврат — не приговор, а данные о том, что ещё организует мой выбор.**

> **Не объяснить себя сценарием, а проверить его следующим ходом и следующей позицией.**

> **Переход начинается не тогда, когда сценарий исчез, а когда он перестаёт быть единственным организатором следующего цикла.**

> **Я больше не живу по сценарию — и снова строю всю жизнь вокруг него.**
