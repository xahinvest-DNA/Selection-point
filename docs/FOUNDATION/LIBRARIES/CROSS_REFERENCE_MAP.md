# Карта перекрёстных связей Selection Point

**Статус:** действующая навигационная библиотека  
**Дата обновления:** 11 июля 2026 года

Карта показывает, какие утверждения являются основанием для следующих элементов и где искать полный смысл.

## Связи модели человека со ступенью 1

| Положение | Основное влияние на ступень 1 |
|---|---|
| SP-HCM-01 — воспринятая модель | SP-S1-P02; различение факта и текущей версии ситуации |
| SP-HCM-02 — состояние | SP-S1-P01–SP-S1-P13; состояние меняет доступность действия |
| SP-HCM-03 — автоматическая цепочка | SP-S1-P01, SP-S1-P05–SP-S1-P13; один эпизод раскладывается на звенья |
| SP-HCM-04 — образ себя и сценарий | SP-S1-P03–SP-S1-P13; роль и идентичность участвуют в поле выбора |
| SP-HCM-05 — действие меняет действующего | SP-S1-P05–SP-S1-P13 и SP-VM-01; ход создаёт следующую позицию |
| SP-HCM-06 — обратная связь | SP-S1-P05–SP-S1-P13; последствия должны вести к корректировке |
| SP-HCM-07 — множественные точки восстановления | SP-S1-P05–SP-S1-P13; позднее замечание не отменяет участие |
| SP-HCM-08 — повторяющиеся циклы | SP-S1-P06–SP-S1-P13 и SP-VM-01; траектория формируется повторением |
| SP-HCM-09 — создание независимо от контроля | делает явным общий вывод: каждый ход и бездействие создают следующую позицию, даже если человек не контролирует исходное событие или результат |

## Внутренняя последовательность первой ступени

```text
SP-S1-P01 — общая позиция внутри автоматизма
→ SP-S1-P02 — переживаемая реальность
→ SP-S1-P03 — переживаемый образ себя
→ SP-S1-P04 — срочная оперативная задача
→ SP-S1-P05 — доступный участок участия
→ SP-S1-P06 — иллюзия неизбежного продолжения
→ SP-S1-P07 — боль расхождения намерения и траектории
→ SP-S1-P08 — реконструкция одного эпизода
→ SP-S1-P09 — тело внутри контура
→ SP-S1-P10 — текущая жизненная позиция
→ SP-S1-P11 — функциональный профиль практики
→ SP-S1-P12 — качественный профиль перехода
→ SP-S1-P13 — ловушка объяснения вместо участия
```

Ступень 1 содержательно завершена.

## Переход от первой ступени ко второй

```text
несколько отдельно реконструированных эпизодов
→ эпизоды сохранены как сопоставимые единицы
→ человек замечает, что некоторые повторения не случайны
→ внешнее сходство ещё смешивается с функциональной организацией
→ SP-S2-P01
```

SP-S1-P12 передаёт второй ступени сопоставимый материал. SP-S1-P13 защищает от ярлыка, объяснения и превосходства вместо участия.

## Связи модели человека со второй ступенью

| Положение | Влияние на SP-S2-P01–SP-S2-P03 |
|---|---|
| SP-HCM-01 | гипотеза о сценарии является воспринятой моделью и может начать определять текущую ситуацию |
| SP-HCM-02 | состояние меняет, какие признаки сценария и роли становятся заметными и какие действия кажутся доступными |
| SP-HCM-03 | сценарий и сценарная роль воспроизводятся через цепочку, а не через один изолированный триггер |
| SP-HCM-04 | образ себя и ожидаемая роль организуют сценарий, но не являются сущностью личности |
| SP-HCM-05 | привычный ход не только выражает роль, но создаёт следующую позицию и укрепляет соответствующий образ себя |
| SP-HCM-06 | последствия могут быть приняты за доказательство сценария и роли либо использованы для их проверки |
| SP-HCM-07 | сценарий и роль могут быть замечены до, во время или после привычного хода; наблюдение не ограничивает участие одной секундой |
| SP-HCM-08 | повторяемые циклы создают устойчивую сценарную реальность и правдоподобность повторяющейся роли |
| SP-HCM-09 | отсутствие контроля над событием или итогом не прекращает создание собственной реальности через восприятие, роль, действие, последствия и следующую позицию |

## Внутренняя последовательность второй ступени

```text
SP-S2-P01
эпизоды различимы, повторение замечается,
но функциональная организация ещё неполно распознана

→ SP-S2-P02
сценарий воспринимается как то, что снова происходит с человеком,
хотя через восприятие, ожидание и действие он создаёт следующую позицию

→ SP-S2-P03
повторяющаяся роль уже замечается,
но человек ещё считает себя носителем, пленником
или неизбежным исполнителем этой роли
```

SP-S2-P04 «Что управляет решениями» не открыт и не выводится автоматически.

## Центральные причинные линии

### SP-HCM-09 и SP-S2-P02

```text
событие
→ сценарное восприятие
→ знакомое значение
→ знакомое ожидание
→ действие или бездействие
→ ответ среды
→ последствия
→ следующая позиция
→ подтверждение знакомой реальности
```

### SP-S2-P03

```text
повторяющийся сценарий
→ повторяющаяся роль
→ «это тот, кто я»
→ действия, совместимые с ролью
→ подтверждение роли
```

Центральные различения:

> Контроль предполагает возможность заранее получить желаемый результат. Создание происходит независимо от контроля.

> Я не являюсь своей ролью, но из этой роли создаю продолжение.

## Связь с SP-VM-01

Для SP-HCM-09:

```text
текущая позиция
→ ход или бездействие
→ следующая позиция
→ реальность следующего цикла
```

Для SP-S2-P02:

```text
сценарное восприятие
→ знакомый ход
→ знакомая следующая позиция
→ подтверждение сценарной реальности
```

Для SP-S2-P03:

```text
позиция, включающая сценарную роль
→ ход, совместимый с ролью
→ следующая позиция
→ подтверждение роли
```

Модель не доказывает полный контроль, создание исходного внешнего события, равную доступность альтернатив или фиксированность сценарной роли.

## Правило ясности будущего слоя доставки

Полная методология хранит причинную цепочку, различения и границы. Будущий учебный материал должен передавать одну центральную линию за один шаг.

Для SP-S2-P03:

```text
повторяющаяся роль
→ «это тот, кто я»
→ действие из роли
→ подтверждение роли
```

Первое учебное различение:

> **Я не являюсь своей ролью, но из этой роли создаю продолжение.**

Полнота архитектуры не требует максимальной плотности каждого учебного элемента.
