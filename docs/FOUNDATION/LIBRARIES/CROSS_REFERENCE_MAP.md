# Карта перекрёстных связей Selection Point

**Статус:** действующая навигационная библиотека  
**Дата обновления:** 7 июля 2026 года

Карта показывает, какие утверждения являются основанием для следующих элементов и где искать полный смысл.

## Связи модели человека со ступенью 1

| Положение | Основное влияние на ступень 1 |
|---|---|
| SP-HCM-01 — воспринятая модель | SP-S1-P02; участвует в реконструкции, различении факта и гипотезы, а в SP-S1-P13 показывает риск превращения объяснения в новую модель, принятую за завершение работы |
| SP-HCM-02 — состояние | SP-S1-P01–SP-S1-P13; влияет на доступность действия и риск подмены участия ясностью, виной, контролем или превосходством наблюдателя |
| SP-HCM-03 — автоматическая цепочка | SP-S1-P01, SP-S1-P05–SP-S1-P13; в P13 объяснение само может стать звеном нового автоматического контура |
| SP-HCM-04 — образ себя и сценарий | SP-S1-P03–SP-S1-P13; в P13 язык системы и способность видеть могут стать статусной идентичностью |
| SP-HCM-05 — действие меняет действующего | SP-S1-P05–SP-S1-P13 и SP-VM-01; объяснение без участия тоже создаёт следующую позицию |
| SP-HCM-06 — обратная связь | SP-S1-P05–SP-S1-P13; ловушка возникает при анализе без обратной связи и изменённого продолжения |
| SP-HCM-07 — множественные точки восстановления | SP-S1-P05–SP-S1-P13; P13 фиксирует потерю восстановления, когда реконструкция не возвращает к доступному участию |
| SP-HCM-08 — повторяющиеся циклы | SP-S1-P06–SP-S1-P13 и SP-VM-01; объяснение может повторяться как новый цикл без перестройки траектории |

## Внутренняя последовательность первой ступени

```text
SP-S1-P01
общая позиция человека внутри автоматизма

→ SP-S1-P02
какую ситуацию он переживает как реальность

→ SP-S1-P03
кем он переживает себя внутри этой ситуации

→ SP-S1-P04
какую срочную задачу должно решить действие

→ SP-S1-P05
где участие в этой цепочке реально доступно

→ SP-S1-P06
какое центральное заблуждение удерживает логику ступени

→ SP-S1-P07
какое повторяющееся расхождение составляет главную боль

→ SP-S1-P08
какая первая способность делает один эпизод различимым

→ SP-S1-P09
как тело участвует в автоматическом контуре

→ SP-S1-P10
как текущая жизненная позиция хранит след предыдущей траектории

→ SP-S1-P11
какие функции должна выполнять практика, чтобы реконструкция завершалась доступным участием

→ SP-S1-P12
по каким качественным признакам операция стала повторяемой, частично самостоятельной и готовой к будущему сопоставлению эпизодов

→ SP-S1-P13
во что освоенная реконструкция может исказиться: объяснение, контроль, самообвинение, правильный язык, диагностика других или превосходство наблюдателя вместо участия
```

Ступень 1 содержательно завершена. Ступень 2 не открыта.

## Связь SP-S1-P13 с SP-VM-01

SP-VM-01 показывает, что объяснение само может стать ходом:

```text
напряжение от повторения
→ объяснение
→ ясность / вина / контроль / превосходство
→ отсутствие участия
→ следующая позиция без изменения траектории
```

Модель не является отдельной практикой и не открывает вторую ступень.

Связанные элементы:

- полный источник: `FIVE_STAGES_THEORY/13_STAGE_1_STAGE_TRAP.md`;
- каноническая карточка: `CANONICAL/05_STAGE_1_P13_CANONICAL.md`;
- новые риски: SP-RISK-026, SP-RISK-027;
- доказательный статус: строка SP-S1-P13 в `EVIDENCE_MAP.md`.


## Правило обновления

После утверждения нового параметра необходимо:

1. добавить связь с его основаниями;
2. указать, что он подготавливает, только если следующий параметр действительно открыт;
3. связать новые термины либо зафиксировать, что новых терминов нет;
4. указать подходящие примеры;
5. зафиксировать применимые риски;
6. связать визуальную модель, если она добавлена или уточнена;
7. не создавать связь с ещё не утверждённым выводом как с фактом.
