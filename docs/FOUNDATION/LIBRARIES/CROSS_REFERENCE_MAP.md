# Карта перекрёстных связей Selection Point

**Статус:** действующая навигационная библиотека  
**Дата обновления:** 6 июля 2026 года

Карта показывает, какие утверждения являются основанием для следующих элементов и где искать полный смысл.

## Внутренняя последовательность первой ступени

```text
SP-S1-P01
общая позиция человека внутри автоматизма

→ SP-S1-P02
какую ситуацию он переживает как реальность

→ SP-S1-P03
кем он переживает себя внутри этой ситуации

→ SP-S1-P04
какую срочную задачу должно решить действие

→ SP-S1-P05
где участие в этой цепочке реально доступно

→ SP-S1-P06
какое центральное заблуждение удерживает логику ступени

→ SP-S1-P07
какое повторяющееся расхождение составляет главную боль

→ SP-S1-P08
какая первая способность делает один эпизод различимым

→ SP-S1-P09
как тело участвует в автоматическом контуре

→ SP-S1-P10
как текущая жизненная позиция хранит след предыдущей траектории

→ SP-S1-P11
какие функции должна выполнять практика, чтобы реконструкция завершалась доступным участием
```

SP-S1-P12 и SP-S1-P13 не выводятся автоматически из SP-S1-P11 и не открыты.

## Граница SP-S1-P11 и Фазы 5

```text
Фаза 5

```

## Связь SP-S1-P11 с SP-VM-01

```text
исходная позиция
→ совершённый ход
→ созданная следующая позиция
→ доступное изменение продолжения
```

SP-VM-01 может поддерживать будущую практическую работу, но не является готовой практикой и не доказывает доступность нарисованных вариантов.

## Правило обновления

После утверждения нового параметра необходимо:

1. добавить связь с его основаниями;
2. указать, что он подготавливает, только если следующий параметр действительно открыт;
3. связать новые термины либо зафиксировать, что новых терминов нет;
4. указать подходящие примеры;
5. зафиксировать применимые риски;
6. связать визуальную модель, если она добавлена или уточнена;
7. не создавать связь с ещё не утверждённым выводом как с фактом.
