## Правила добавления

1. Новая строка со статусом `утверждено` появляется только после явного решения Андрея.
2. У утверждения должен существовать полный теоретический источник или отдельный утверждённый документ модели.
3. Одновременно создаётся или обновляется каноническая карточка.
4. Формулировка в реестре должна быть краткой и не добавлять новый смысл.
5. При пересмотре положение не удаляется молча: статус меняется на `требует пересмотра`, а причина фиксируется.
6. Черновая гипотеза не получает ID утверждённого положения.
