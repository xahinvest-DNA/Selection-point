# 2. Центральное основание Selection Point

## SP-HCM-09

> **Контроль предполагает возможность заранее получить желаемый результат. Создание происходит независимо от контроля.**

Человек не создаёт все внешние события, но осознанно или автоматически участвует в создании собственной последующей жизненной позиции из того, что с ним происходит:

```text
событие
→ восприятие
→ значение
→ ожидание
→ действие или бездействие
→ ответ среды
→ последствия
→ следующая позиция
```

Обязательные различения:

```text
создание собственной реальности
≠ создание всех внешних событий

создание
≠ полный контроль

ответственность
≠ моральная вина

отсутствие гарантированного результата
≠ отсутствие авторства
```

# 3. Завершённая архитектура первой ступени

Ступень 1 «Жизнь внутри автоматизма» утверждена полностью: SP-S1-P01–SP-S1-P13.

Итог первой ступени:

```text
один автоматический эпизод становится различимым
→ его звенья восстанавливаются
→ доступное участие обнаруживается
→ хотя бы одно продолжение изменяется
→ эпизод сохраняется как сопоставимый материал
→ ловушка объяснения вместо выбора различается
```

# 4. Действующая архитектура второй ступени

Утверждены SP-S2-P01–SP-S2-P10.

Ключевая последовательность:

```text
сопоставимые эпизоды
→ замеченное повторение
→ сценарное восприятие реальности
→ сценарная роль
→ сценарная задача
→ доступная точка, где требованию можно не подчиниться
→ различение самостоятельного направления и противоположности прошлому
→ боль возврата после понимания
→ фактическая следующая позиция становится обратной связью
→ обнаруживается повторяющаяся телесная организация
→ обнаруживается повторяющаяся функциональная жизненная позиция
```

## SP-S2-P08 — Следующий навык

> **Возврат — не приговор, а данные о том, что ещё организует мой выбор.**

## SP-S2-P09 — Телесные проявления

> **Ситуации меняются, а тело снова занимает знакомую позицию.**

> **Повторение знакомой телесной реакции ещё не означает повторения всей сценарной траектории.**

## SP-S2-P10 — Жизненные проявления

**Рабочее название:** «Обстоятельства меняются, а я снова занимаю знакомое место».

> **Разные ситуации — та же функциональная позиция.**

> **Изменение внешних обстоятельств ещё не означает изменения сценарной организации, а повторение внешнего результата ещё не доказывает повторения сценария.**

Центральная конструкция:

```text
жизненная ситуация
→ знакомая роль
→ сценарная задача
→ распределение времени, денег,
  внимания, близости, риска и обязательств
→ действие или бездействие
→ материальная и социальная следующая позиция
→ последствия закрепляются в устройстве жизни
→ новое поле ходов
```

Ключевые различения:

```text
повторяющийся жизненный результат
≠ доказанный повторяющийся сценарий

последствие прошлого сохраняется
≠ механизм активно воспроизводится

участие в траектории
≠ создание всех исходных событий

повторение в одной сфере
≠ сценарий всей личности
```
