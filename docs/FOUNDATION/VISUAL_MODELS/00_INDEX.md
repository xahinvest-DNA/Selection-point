# Визуальные модели Selection Point

**Статус:** действующий фундаментальный вспомогательный слой  
**Дата введения:** 5 июля 2026 года  
**Дата обновления:** 11 июля 2026 года

## Назначение

Папка хранит утверждённые сквозные визуальные модели, связывающие положения Selection Point.

Визуальная модель:

- не заменяет полный теоретический источник;
- не является самостоятельным доказательством;
- не добавляет новый смысл;
- содержит границы метафоры;
- получает стабильный ID;
- связывается с утверждёнными положениями и терминами.

## SP-VM-01 — Граф траекторий выбора

Материалы:

- `01_CHOICE_TRAJECTORY_GRAPH.md`;
- `assets/choice_trajectory_graph.svg`.

Модель связана с:

- SP-HCM-05–SP-HCM-08;
- SP-S1-P05 — доступная точка выбора;
- SP-S1-P06 — иллюзия неизбежного продолжения;
- SP-S1-P07 — расхождение значимого направления и фактической траектории;
- SP-S1-P08 — реконструкция одного эпизода как последовательности позиций и ходов;
- SP-S1-P09 — телесное состояние как часть исходной позиции, перехода и следующей позиции;
- SP-S1-P10 — текущая жизненная позиция как накопленная конфигурация внешних обстоятельств, предыдущих ходов и последствий;
- SP-S1-P11 — связь совершённого хода, созданной позиции и доступного изменения продолжения;
- SP-S1-P12 — сохранение нескольких отдельных последовательностей как сопоставимых эпизодов;
- SP-S1-P13 — риск того, что объяснение само становится ходом, создающим следующую позицию без изменения траектории;
- SP-S2-P01 — несколько отдельных последовательностей становятся материалом для осторожного сопоставления без доказательства единого сценария.

Для SP-S1-P13 модель поддерживает только утверждённую функцию:

```text
напряжение от повторения
→ объяснение / диагностика / чувство превосходства
→ временная ясность или контроль
→ отсутствие собственного доступного участия
→ следующая позиция почти не изменилась
```

Для SP-S2-P01 модель поддерживает только следующую функцию:

```text
эпизод A: позиция → ход → следующая позиция
эпизод B: позиция → ход → следующая позиция
эпизод C: позиция → ход → следующая позиция

несколько отдельных последовательностей
→ материал для будущего сопоставления
```

SP-VM-01 не определяет:

- какие эпизоды относятся к одному повторяющемуся сценарию;
- функциональное тождество внешне похожих действий;
- способ сравнения эпизодов;
- SP-S2-P02 и следующие параметры второй ступени;
- практику выхода из ловушки;
- оценочную процедуру;
- курс или интерфейс.

Отдельная визуальная модель для SP-S2-P01 не требуется.

Практическая, учебная и оценочная формы SP-VM-01 создаются только на соответствующих последующих фазах.
