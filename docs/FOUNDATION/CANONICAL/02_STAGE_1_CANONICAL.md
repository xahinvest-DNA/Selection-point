# SP-S1-P01 — Состояние человека

## Утверждённое положение

Человек находится внутри автоматической реакции и недостаточно различает её звенья; первым движением может стать исследовательский интерес к одному конкретному эпизоду.

## Основная конструкция

```text
условия и состояние
→ направление внимания
→ значение ситуации
→ активированный сценарий
→ оперативная задача
→ импульс
→ действие
```

## Короткая формула

> **Человек не наблюдает автоматизм — он живёт изнутри него.**

---

# SP-S1-P02 — Способ восприятия реальности

## Утверждённое положение

Человек смотрит из текущей модели ситуации и принимает её за саму реальность; первое расширение — различить событие и текущую версию события.

## Основная конструкция

```text
событие
+ избирательное внимание
+ состояние
+ значение
+ предполагаемая причина
+ прогноз
+ очевидное действие
= переживаемая реальность
```

## Короткая формула

> **Первый разрыв возникает между «так всё и есть» и «так я сейчас это воспринимаю».**

---

# SP-S1-P03 — Способ восприятия себя

## Утверждённое положение

Человек принимает активированный образ себя за сущность, а роль — за границу допустимых действий; первый разрыв локализует образ в конкретном эпизоде.

## Основная конструкция

```text
событие и состояние
→ значение ситуации
→ активированный образ себя
→ ожидаемый сценарий
→ границы допустимого
→ действие, совместимое с ролью
```

## Короткая формула

> **Знакомая роль становится границей допустимых действий.**

---

# SP-S1-P04 — Что управляет решениями

## Утверждённое положение

Решением преимущественно управляет доминирующая оперативная задача текущего эпизода, а приоритет получает знакомое и психологически доступное средство.

## Основная конструкция

```text
состояние
→ воспринятая ситуация
→ образ себя
→ угроза, потребность или дефицит
→ оперативная задача
→ психологически доступные средства
→ знакомое действие
```

## Короткая формула

> **Импульс показывает, что хочется сделать; оперативная задача — что это действие должно дать.**

---

# SP-S1-P05 — Доступная точка выбора

## Утверждённое положение

Доступная точка возникает на первом различимом участке цепочки, где человеку доступен хотя бы один выполнимый ход, изменяющий дальнейшую траекторию.

## Три критерия доступности

1. **Различимость:** человек способен заметить хотя бы один фрагмент цепочки.
2. **Выполнимость:** существует хотя бы один ход, возможный в реальном состоянии.
3. **Изменение траектории:** ход меняет время, масштаб, форму, ущерб, восстановление или условия следующего цикла.

## Основная конструкция

```text
текущая позиция
→ поле доступных ходов
→ следующий ход
→ новая позиция
→ новое поле ходов
```

## Короткие формулы

> **Доступная точка выбора — первый замеченный участок цепочки, где следующий ход ещё можно изменить.**

> **Выбор первой ступени — не полный контроль, а один дополнительный реально выполнимый ход.**

> **Каждый ход создаёт поле следующего хода.**

# SP-S1-P06 — Главная иллюзия

## Рабочее название

«Иллюзия неизбежного продолжения».

## Утверждённое положение

Текущая внутренняя сборка ситуации переживается как объективная необходимость, поэтому привычное продолжение принимается за единственно возможную траекторию.

## Основная конструкция

```text
состояние сужает внимание
→ интерпретация становится реальностью
→ образ себя становится сущностью
→ оперативная задача становится необходимостью
→ привычное средство становится единственным доступным
→ действие переживается как неизбежное
```

## Короткая формула

> **Такова ситуация, таков я — значит, дальше иначе быть не может.**

# SP-S1-P07 — Главная боль

## Рабочее название

«Боль повторяющегося расхождения между намерением и траекторией».

## Утверждённое положение

Повторяющееся расхождение между значимым направлением и фактически создаваемой траекторией становится центральной болью, когда локально необходимые ходы снова приводят к нежелательному результату.

## Основная конструкция

```text
значимое направление
→ состояние и оперативная задача
→ привычный ход переживается как необходимый
→ немедленный результат
→ отложенная цена
→ возвращение широкого горизонта
→ обнаружение расхождения
→ самообвинение, жёсткое обещание или бессилие
→ новый цикл
```

## Короткая формула

> **Каждый шаг казался необходимым — но снова привёл меня туда, куда я не хотел.**

---

# SP-S1-P08 — Следующий навык

## Рабочее название

«Реконструкция конкретного автоматического эпизода».

## Утверждённое положение

Человек способен реконструировать один конкретный автоматический эпизод как последовательность различимых звеньев и обнаружить первый реально доступный участок изменения дальнейшего продолжения.

## Основная конструкция

```text
конкретный результат
→ выбор одного эпизода
→ различение условий и состояния
→ различение воспринятой ситуации и образа себя
→ гипотеза об оперативной задаче
→ привычный ход и немедленный результат
→ отложенная цена и следующая позиция
→ первый реально доступный участок изменения продолжения
```

## Короткая формула

> **Не «почему я опять такой», а «как именно это произошло и где продолжение ещё может измениться».**

---

# SP-S1-P09 — Телесные проявления

## Рабочее название

«Тело внутри автоматического контура».

## Утверждённое положение

Тело включено во весь автоматический контур, но недостаточно различается как исходное состояние, носитель реакции, объект регулирования, получатель немедленного результата, носитель отложенной цены и часть следующей позиции.

## Основная конструкция

```text
условия жизни
→ телесное состояние
→ изменение внимания и восприятия
→ срочная оперативная задача
→ привычный способ изменить состояние
→ немедленный телесный результат
→ отложенная телесная цена
→ следующее состояние
→ новый автоматический эпизод
```

## Короткая формула

> **Состояние становится командой, а последствия привычного хода — состоянием следующего цикла.**

---

# SP-S1-P10 — Жизненные проявления

## Рабочее название

«Текущая жизнь как позиция траектории».

## Утверждённое положение

Текущая жизненная позиция является фактом, в котором встречаются внешние обстоятельства, действия других людей, предыдущие собственные ходы, бездействие и накопленные последствия. Трудность может стать точкой исследования предыдущей и следующей траектории без превращения её в вину или предопределённый смысл.

## Основная конструкция

```text
текущая жизненная позиция
=
объективные обстоятельства
+ действия других людей
+ предыдущие собственные ходы
+ бездействие
+ накопленные последствия

увидеть факт
→ различить формирование позиции
→ отделить внешнее от собственной части участия
→ увидеть отношение и действие сейчас
→ создать следующую позицию
```

## Короткая формула

> **Текущая жизнь — не приговор и не обвинение, а позиция, в которой видна предыдущая траектория и начинается следующая.**

# Завершение первой ступени

Функциональный профиль практик, качественный профиль перехода и ловушка первой ступени зафиксированы отдельно:

- SP-S1-P11: `03_STAGE_1_P11_CANONICAL.md`;
- SP-S1-P12: `04_STAGE_1_P12_CANONICAL.md`;
- SP-S1-P13: `05_STAGE_1_P13_CANONICAL.md`.

Итоговая последовательность первой ступени:

```text
один автоматический эпизод становится различимым
→ его звенья восстанавливаются
→ доступное участие обнаруживается
→ хотя бы одно продолжение изменяется
→ эпизод сохраняется как сопоставимый материал
→ ловушка объяснения вместо выбора различается
```

Переход ко второй ступени не означает исчезновения автоматизма. Он становится обоснованным, когда несколько отдельно реконструированных эпизодов удерживаются как сопоставимые единицы и могут исследоваться без преждевременного личностного приговора или назначения единого сценария.
