# Дополнение к контрольной точке операционного протокола

**Статус:** действующее дополнение к `PROJECT_OPERATING_PROTOCOL.md`  
**Дата:** 7 июля 2026 года  
**Основание:** явное утверждение Андреем SP-S1-P12

Это дополнение обновляет текущую контрольную точку. Общие правила, роли, иерархия источников, порядок фаз и рабочий цикл остаются без изменений.

## Актуальная контрольная точка

- Активная фаза: Фаза 3 — точная архитектура пяти ступеней.
- Активная ступень: Ступень 1 — «Жизнь внутри автоматизма».
- SP-S1-P01–SP-S1-P12 утверждены.
- Последнее утверждённое положение: SP-S1-P12 «Критерии перехода».
- SP-S1-P13 «Ловушка ступени» не открыт.

## Утверждённая короткая формула SP-S1-P12

> **Переход начинается не тогда, когда автоматизм исчез, а когда отдельные повторения стали различимыми, изменяемыми и пригодными для сопоставления.**

## Утверждённая межфазная граница

В Фазе 3 SP-S1-P12 определяет качественный профиль готовности. Измеримые пороги и процедура подтверждения создаются только в Фазе 6.
