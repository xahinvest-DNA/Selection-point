# PRACTICE_SYSTEM

## Purpose

This document defines the practice architecture behind Selection Point.

The first mobile app version should use simplified versions of these practices.

Do not expose advanced nondual language to first-time users. Translate practices into body, pause, impulse, energy and next action.

---

# 1. Practice layers

The system contains three practical layers for v1.

## Layer 1 — Stop automatic reaction

Goal: interrupt old programs before they become action.

Core tools:

- distinction;
- expansion;
- stop-frame;
- body contact;
- urge as object, not command.

User-facing language:

```txt
Stop. Notice what is happening. Feel the body. Do not obey the urge immediately.
```

## Layer 2 — Trust the small impulse

Goal: after automatic reaction is interrupted, detect a small natural movement.

Core tools:

- 3-breath check;
- compression vs expansion;
- micro-action;
- delay test.

User-facing language:

```txt
After the pause, what is the smallest action that feels light or neutral?
```

## Layer 3 — Current life visibility

Goal: allow the user to experience the current moment without turning it into analysis.

Core tools:

- short direct prompts;
- silence after prompt;
- non-evaluation;
- return to ordinary action.

User-facing language:

```txt
Read. Pause. Notice. Continue.
```

# 2. Micro-protocol for urge

Use for food impulse, alcohol impulse, phone scrolling, anger, avoidance.

```txt
1. Stop phrase: “30 seconds.”
2. Breath: feel one inhale and one exhale.
3. Body: feel feet, chair or hands.
4. Sound: hear one sound around you.
5. Include the urge: where is it in the body?
6. Ask: did it change?
7. Choose: no action, small action, or conscious action.
```

Important: the app must not frame this as success/failure.

Even if the user follows the urge, the useful result is data:

```txt
What triggered it?
What did it feel like?
Was there a pause?
What did the pause change?
```

# 3. Micro-protocol for avoidance / laziness

Use for household tasks, movement, exercise, planning, work.

```txt
1. Notice phrase: “I am in the image of the task, not in the task.”
2. Feel body in current position.
3. Add sounds and space around.
4. Include resistance as a body sensation.
5. Do not decide to do the whole task.
6. Choose one movement only:
   - stand up;
   - take one object;
   - open the door;
   - put shoes on;
   - move one item;
   - walk for one minute.
7. Stop or continue naturally.
```

User-facing principle:

```txt
Do not start the whole task. Start one visible movement.
```

# 4. Micro-protocol for evening overeating

```txt
1. Name the moment: “evening impulse.”
2. Pause before eating.
3. Ask: hunger, fatigue, anxiety, reward or emptiness?
4. Feel body for 3 breaths.
5. If hunger: eat a planned meal.
6. If fatigue: water + 5 minutes rest or walk.
7. If anxiety: write one sentence: “Right now I am trying not to feel…”
8. Record what happened without shame.
```

# 5. Daily 2-minute morning direction

Purpose: set a flexible Point B without self-violence.

```txt
1. Three calm breaths.
2. What is Point A this morning?
3. What direction feels useful today?
4. What is one small action that does not create heaviness?
5. Commit only to that action.
```

Example:

```txt
A: slept badly, body heavy.
B: lighter day.
Small action: 10 minutes walk after first meal.
```

---

# 6. Daily 2-minute evening review

Purpose: extract feedback without guilt.

```txt
1. What was one moment of automatic reaction today?
2. Was there any pause?
3. What changed after the pause?
4. What was one natural impulse I followed?
5. What is tomorrow’s smallest useful direction?
```

No moral score. Only observation.

# 7. Direct pointing practice — short version

Use only as optional advanced practice, not as core MVP.

Duration: 5–7 minutes.

```txt
1. Close your eyes. Exhale. Do nothing.
2. Find the breath. Do not control it.
3. Where is the border between the one who breathes and breathing?
4. A thought appeared. Did you call it?
5. The next thought: where did it come from?
6. Try to find the one who is looking.
7. Return to breath, sounds, body.
8. Open your eyes. Do not evaluate.
```

Purpose: create a pause before identity-driven reaction.

---

# 8. Direct pointing practice — extended version

Use only after the user has stable grounding and explicitly opts in.

Duration: 12 minutes.

```txt
1. Close eyes. Exhale. Let the body slow down.
2. Feel breathing without changing it.
3. Who knows the breath?
4. A thought appeared. Did you choose it?
5. The next thought: where did it arise from?
6. Find the body sensation called “I”.
7. Look at that “I” as an object.
8. Who looks at the feeling of “I”?
9. Return to breath.
10. Return to sounds.
11. Return to body.
12. Everything at once: breath, sounds, body.
13. Find the boundary between inside and outside.
14. If a thought says “it works” or “it does not work”, see it as another sound.
15. Stop.
16. Return to room.
17. Open eyes. Move fingers.
```

Safety note: this practice can be disorienting. It must not be the default path.

# 9. Marker tracking

The app should avoid long self-analysis in v1.

Use short markers:

```txt
+  shift happened
0  no visible change
-  worse / more tense / more empty / more strange
```

Use in:

- urge pause;
- avoidance pause;
- evening overeating protocol;
- direct pointing practice;
- daily review.

---

# 10. What the app should train first

Priority order for v1:

1. Stop before automatic action.
2. Feel body and environment.
3. Name the mechanism.
4. Choose one small movement.
5. Record feedback.
6. Repeat.

Not v1:

- deep nondual exploration;
- “no-self” language;
- advanced level systems;
- spiritual claims;
- promises of enlightenment;
- medical or addiction treatment claims.
