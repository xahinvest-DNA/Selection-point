# Ключевые понятия

## 1. Точка выбора

Точка выбора — это момент между стимулом и действием.

Примеры:

- захотелось съесть сладкое после тяжелого дня;
- появилось желание налить алкоголь;
- надо выйти на прогулку, но тело будто выключается;
- собеседник сказал неприятное, и хочется резко ответить;
- есть домашняя задача, но внутри возникает "не сейчас".

Обычно человек сразу становится импульсом. Selection Point тренирует паузу до действия.

Пользовательская формулировка:

```txt
Я замечаю момент до автоматического хода.
```

## 2. Точка А

Точка А — это текущий факт.

Она описывает, что есть сейчас:

- вес;
- сон;
- энергия;
- питание;
- движение;
- алкоголь;
- пищевой импульс;
- переедание;
- триггер;
- состояние тела;
- незавершенная задача.

Точка А не содержит моральной оценки.

Факт:

```txt
Я ел после 22:00.
```

История:

```txt
Я безвольный.
```

## 3. История

История — это то, что ум добавил к факту.

Истории часто звучат так:

- "я опять всё испортил";
- "у меня никогда не получится";
- "я должен срочно исправиться";
- "если я сегодня не сделал идеально, всё бессмысленно";
- "нормальные люди так не делают";
- "я слабый".

История усиливает напряжение и часто запускает новый автоматизм.

## 4. Точка Б

Точка Б — это полезное направление.

Она не должна быть жесткой целью. Жесткая цель легко превращается в давление.

Жесткая точка Б:

```txt
За 14 дней я обязан похудеть на 5 кг.
```

Гибкая точка Б:

```txt
Я хочу сделать вечера спокойнее и увидеть, что запускает переедание.
```

Точка Б должна ощущаться как направление, которое можно тестировать.

## 5. Разность потенциалов

Разность потенциалов — это напряжение между точкой А и точкой Б.

Пример:

```txt
А: вечером я переедаю после усталого дня.
Б: спокойный вечер без автоматического переедания.
Разность: я не умею проживать вечернюю усталость без еды.
```

Из этой разности может возникнуть внутренняя задача.

## 6. Автоматический механизм

Автоматический механизм — повторяющаяся реакция, которая запускается раньше ясного выбора.

Примеры:

- усталость -> сладкое;
- тревога -> алкоголь;
- скука -> телефон;
- большая задача -> избегание;
- стыд -> жесткий контроль;
- срыв -> вина -> новый срыв;
- конфликт -> защита или нападение.

Механизм — не дефект характера. Это повторяемая цепочка.

## 7. Второй узел

Первый узел — реальная разность:

```txt
есть грязная посуда -> можно сделать кухню чище
```

Второй узел — давление вокруг этой разности:

```txt
я должен;
я ленивый;
я опять не справился;
пока не сделаю всё, я плохой.
```

Второй узел забирает больше энергии, чем сама задача.

Selection Point учит замечать второй узел и не строить действие из него.

## 8. Пауза

Пауза — короткий разрыв между импульсом и действием.

Она может длиться 30 секунд.

Пауза не обязана остановить действие полностью. Даже если пользователь всё равно поел, выпил или отложил дело, пауза дала данные:

- что было триггером;
- где импульс ощущался в теле;
- изменился ли импульс;
- была ли возможность другого хода.

## 9. Расширение

Расширение — это включение тела и среды.

Простой формат:

```txt
дыхание + контакт тела + звуки + текущий импульс
```

Расширение не подавляет импульс. Оно делает его одним объектом в поле, а не приказом.

## 10. Внутренняя задача

Внутренняя задача — один тренируемый переход.

Не:

```txt
стать дисциплинированным человеком
```

А:

```txt
в момент вечерней усталости сделать 30-секундную паузу до еды и понять: это голод, тревога или потребность в отдыхе.
```

Хорошая внутренняя задача:

- конкретна;
- проверяема;
- мала;
- связана с повторяющимся механизмом;
- не содержит самонаказания.

## 11. Импульс и автоматическая реакция

Автоматическая реакция обычно ощущается как:

- сжатие;
- давление;
- срочность;
- "надо";
- "быстрее";
- "иначе будет плохо".

Живой импульс чаще ощущается как:

- легкое движение;
- нейтральная ясность;
- "можно попробовать";
- отсутствие драмы;
- возможность подождать.

Пользователь не обязан идеально различать их с первого дня. Это тренируется.

## 12. Маленькое действие

Маленькое действие — следующий видимый ход, а не вся задача.

Примеры:

- встать;
- надеть обувь;
- налить воды;
- записать одну строку;
- открыть дверь;
- убрать один предмет;
- пройти одну минуту;
- приготовить один нормальный прием пищи.

Маленькое действие снижает сопротивление, потому что не требует решить всю жизнь сразу.

## 13. Обратная связь

Обратная связь — то, что показала реальность.

Не оценка:

```txt
хорошо / плохо
```

А наблюдение:

```txt
после паузы импульс ослаб;
после недосыпа тяга к сладкому была сильнее;
если ужин был заранее, переедания не было;
после алкоголя энергия утром 2 из 10.
```

## 14. Маркеры + / 0 / -

Короткая фиксация после практики:

- `+` — сдвиг был;
- `0` — заметного изменения нет;
- `-` — стало хуже, напряженнее, пусто или странно.

Это не оценка пользователя. Это технический маркер состояния.
