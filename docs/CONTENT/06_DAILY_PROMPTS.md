# Ежедневные сообщения

Эти сообщения можно использовать в ручном пилоте: отправлять утром и вечером в мессенджере. В приложении они позже могут стать уведомлениями, карточками дня или текстами чек-ина.

Тон: спокойно, коротко, без давления.

## Универсальное утреннее сообщение

```txt
Доброе утро.

Сегодня не нужно исправлять всю жизнь.

1. Какая точка А сейчас: тело, энергия, сон?
2. Какое направление сегодня полезно?
3. Какое одно маленькое действие не создает тяжесть?

Запиши только это.
```

## Универсальное вечернее сообщение

```txt
Вечерний обзор без оценки.

1. Где сегодня включился автоматический механизм?
2. Была ли пауза хотя бы на несколько секунд?
3. Что показала реальность?
4. Какой один вывод на завтра?

Это данные, не приговор.
```

## День 0

Утро/старт:

```txt
Сегодня стартовая диагностика.

Не меняй себя. Просто зафиксируй точку А:
вес, талия, сон, энергия, питание, движение, алкоголь, главная трудность.

Потом выбери точку Б как направление, а не жесткую цель.
```

Вечер:

```txt
Что стало видно уже после стартовой диагностики?

Запиши одну фразу:
"Главный повтор, который я хочу изучить: ..."
```

## День 1

Утро:

```txt
Сегодня практика: факт без истории.

Когда заметишь трудность, раздели:
факт — что произошло;
история — что ум добавил.
```

Вечер:

```txt
Какой факт ты сегодня заметил?
Какую историю ум добавил сверху?

Не исправляй. Просто раздели.
```

## День 2

Утро:

```txt
Сегодня ищем момент до действия.

Когда появится тяга, сопротивление или "не сейчас", остановись на 30 секунд.
```

Вечер:

```txt
Был ли сегодня импульс?
Где он ощущался в теле?
Изменился ли после паузы?
```

## День 3

Утро:

```txt
Сегодня собираем повтор.

Замечай цепочку:
триггер -> состояние -> импульс -> действие -> последствия.
```

Вечер:

```txt
Что повторялось за эти 3 дня?
Не делай вывод о себе.
Назови механизм.
```

## День 4

Утро:

```txt
Сегодня выбираем один главный механизм.

Не всё сразу. Один повтор, который сильнее всего влияет на тело и энергию.
```

Вечер:

```txt
Заполни:
Когда ..., я обычно ...
Следующие 7 дней я тренирую ...
```

## День 5

Утро:

```txt
Сегодня задача простая: пауза перед главным механизмом.

Даже если действие потом случится, пауза уже дает данные.
```

Вечер:

```txt
Был ли главный механизм?
Была ли пауза?
Что она изменила?
```

## День 6

Утро:

```txt
Сегодня добавляем расширение.

Когда появится импульс:
дыхание + тело + звуки + сам импульс.
```

Вечер:

```txt
Что было до расширения?
Что было после?
Поставь маркер: + / 0 / -.
```

## День 7

Утро:

```txt
Сегодня первый недельный вывод.

Не оценивай результат. Ищи данные.
```

Вечер:

```txt
Что стало понятнее?
Что усиливает механизм?
Что ослабляет механизм?
```

## День 8

Утро:

```txt
Сегодня система вместо силы воли.

Что можно подготовить заранее, чтобы вечером не бороться с собой?
```

Вечер:

```txt
Какое одно изменение среды ты сделал?
Что оно изменило?
```

## День 9

Утро:

```txt
Сегодня одно видимое движение.

Не начинай всю задачу. Сделай один шаг, который можно увидеть глазами.
```

Вечер:

```txt
Какой микро-шаг был сегодня?
Он остановился на одном шаге или продолжился?
```

## День 10

Утро:

```txt
Сегодня ищем второй узел.

Где к задаче добавляется "я должен", "я слабый", "я опять"?
```

Вечер:

```txt
Какая фраза давления появилась?
Что осталось бы от задачи без этой фразы?
```

## День 11

Утро:

```txt
Сегодня проверка импульса.

Перед действием спроси:
это сжатие или расширение?
это толкает или приглашает?
```

Вечер:

```txt
Какой импульс ты проверил?
Что показало тело?
```

## День 12

Утро:

```txt
Сегодня смотрим на всю картину.

Какие закономерности уже очевидны?
```

Вечер:

```txt
Назови 1-3 повторения:
сон, энергия, еда, алкоголь, движение, избегание.
```

## День 13

Утро:

```txt
Сегодня формулируем следующую внутреннюю задачу.

Не крупную цель. Один тренируемый переход.
```

Вечер:

```txt
Когда ..., вместо автоматического ..., я тренирую ...
```

## День 14

Утро:

```txt
Сегодня завершение первого цикла.

Это не экзамен. Это новая точка А.
```

Вечер:

```txt
Что ты теперь видишь о своем механизме?
Что стоит продолжить?
Какая следующая точка Б?
```
