# Индекс контента — LEGACY

**Статус:** ранний контентный черновик.  
**Не является действующей архитектурой курса, практик или продукта.**

Эта папка была создана до утверждения новой архитектуры Selection Point. Она содержит полезные формулировки, примеры, практики и продуктовые гипотезы, но каждый элемент должен быть заново проверен через `docs/FOUNDATION/`.

## Почему папка переведена в legacy

Материалы заранее фиксировали:

- 15-блочную последовательность;
- 14-дневный протокол;
- набор ежедневных сообщений;
- библиотеку практик;
- тексты будущих экранов;
- структуру ручного пилота;
- прежний методический каркас Point A / Point B / potential difference.

Действующий операционный протокол запрещает считать эти формы утверждёнными до завершения архитектуры пяти ступеней, практической системы, критериев освоения и решения о формате первой версии.

## Как использовать материалы

Каждый фрагмент из `docs/CONTENT/` может получить один из статусов:

1. **подтверждено новой архитектурой** — может быть перенесён после проверки формулировок;
2. **полезная гипотеза** — сохраняется для будущего обсуждения;
3. **преждевременный или противоречащий материал** — остаётся в архиве и не влияет на текущую работу.

Нельзя ссылаться на материал этой папки как на принятое решение только потому, что он уже написан.
